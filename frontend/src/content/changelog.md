# Novedades

## v0.2.0 — 2026-05-07

- **Acceso curado**: cada usuario admin ve sólo lo que necesita. Menos ruido, más foco.
- **Agente Anita**: chat por voz y texto, propuestas pendientes con un toque para aceptar.
- **WhatsApp en vivo**: inbox de clientes con conversaciones bidireccionales.
- **Escáner de documentos**: cámara con overlay de carnet, filtros mágicos, PDF multi-página.
- **Modo offline**: marcá un documento con la estrella para tenerlo disponible sin internet.

## v0.1.0 — 2026-04-30

- Lanzamiento de la base de PropOS post-cleanup. Identidad, roles y plataforma multi-tenant.
